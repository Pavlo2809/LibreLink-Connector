# Changelog

All notable changes to LibreLink Connector will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-20

### Added
- Initial release of LibreLink Connector
- Modern WPF UI for displaying glucose readings
- Integration with LibreLinkUp API (EU and Global servers)
- Secure credential storage using Windows Data Protection API
- Auto-login functionality with "Remember Me" option
- Automatic data refresh with configurable intervals (1-30 minutes)
- Manual refresh button for on-demand updates
- Historical glucose data display with recent readings
- Trend arrows indicating glucose direction (rising, falling, stable)
- Color-coded status indicators (Normal, High, Low)
- Target range display (high/low thresholds)
- Configurable settings:
  - Update interval
  - Server region selection
  - Custom alert thresholds
- Light and dark theme support
- Comprehensive error handling and logging
- Patient information display
- Last update timestamp
- API logging for troubleshooting

### Features
- ✅ Secure encrypted credential storage
- ✅ Auto-update at configurable intervals
- ✅ Clean, modern user interface
- ✅ Multiple server region support
- ✅ Trend visualization
- ✅ Historical data view
- ✅ Customizable settings
